# Future Features Roadmap

This document tracks high-impact features to make **unrot** more fun, shareable, and monetizable.

## Priority Tiers

- **P0**: Build first (highest impact for growth + paid conversion)
- **P1**: Build next (retention + stronger monetization)
- **P2**: Nice-to-have expansion

---

## P0 — Best 3 Features to Start Now

## 1) Shareable Session Cards + Vertical Recap (P0)
**Why this matters:** Gives users a one-tap reason to post on TikTok/IG and creates organic acquisition.

### MVP scope
- Generate a clean share card from each completed session:
  - user name
  - focus score
  - depth
  - time focused
  - top 3 topics in chain
- Add a “Share” CTA on Summary + Results pages.
- Add export presets for 9:16 and 1:1.

### Success metric
- Share rate per completed session.
- New users attributed to shared links.

---

## 2) Daily Streak + Quest System (P0)
**Why this matters:** Makes usage habit-forming and gives users daily motivation.

### MVP scope
- Daily streak counter (with grace logic).
- 1 daily quest (e.g., “Reach depth 3” or “Focus 12 minutes”).
- Reward: badge XP + profile flair.
- Display streak/quest in navbar, profile, and summary.

### Success metric
- D1/D7 retention lift.
- Sessions per active user per week.

---

## 3) Pro Smart Paths (Personalized AI Learning Paths) (P0)
**Why this matters:** Strong paid reason to upgrade beyond static content.

### MVP scope
- User picks goal (e.g., “Learn quantum fundamentals”).
- AI generates a 5-topic path from existing topics.
- Pro users can regenerate or extend paths.
- Save path progress to profile.

### Success metric
- Free → Pro conversion rate.
- Pro engagement (sessions started from smart paths).

---

## P1 — Next Wave

## 4) Focus Battles (Async challenge mode)
- Challenge friends with same topic/time cap.
- Winner by score/depth consistency.

## 5) Advanced Pro Analytics
- Best focus hours, depth trend, score trend, topic category heatmap.

## 6) Seasonal Events + Limited Badges
- Monthly focus tournaments and collectible badges.

## 7) Team/Circle Leaderboards
- Private groups with accountability and shared progress.

---

## P2 — Expansion

## 8) Creator Mode
- Public profile pages with “featured paths” and share templates.

## 9) Native Mobile App Companion
- Push reminders, streak rescue, quick session start.

## 10) Offline Reading Packs
- Download topic packs for focus sessions without internet.

---

## Suggested Build Order (Practical)
1. Shareable Session Cards
2. Streak + Quest
3. Pro Smart Paths
4. Advanced Pro Analytics
5. Focus Battles

---

## Notes
- Keep each feature MVP small and shippable in 1–2 weeks.
- Always pair feature launches with one shareable content mechanic.
- Track feature adoption in admin analytics from day one.
